How to use this bot:
    1. Open twitter-dm-bot-win.exe (If on windows).
    2. A chrome window will open, there will be a page with a text box
       where you can type the message you want to send. Remember when you
       hit submit there is no fixing the message, you must restart the bot.
    3. After hitting submit you will be prompted to log in to twitter.
    4. Once you log in the bot will begin sending the message to everyone in
       your clients list spreadsheet.
    5. When the bot finishes sending messages it will close.
